# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-18

### Added
- Initial production release of Radikari Chat Widget
- Web component built with LitElement and TypeScript
- Real-time streaming chat functionality
- Server-Sent Events support for live responses
- Origin validation and tenant isolation
- Session storage for thread persistence
- Responsive design with mobile support
- Customizable theming via CSS properties
- ES modules and UMD bundle support
- Comprehensive documentation and deployment guides
- Semantic versioning implementation
- CDN deployment support (jsDelivr, unpkg)
- Development and production build configurations
- Error handling and recovery mechanisms
- Multi-language support (Indonesian and English)

### Features
- **Chat Functionality**
  - Thread creation and management
  - Message streaming with real-time updates
  - Automatic thread recovery on expiry
  - Message history persistence

- **Integration**
  - Framework-agnostic web component
  - Simple HTML attribute configuration
  - CDN and self-hosting options
  - Legacy browser support via UMD bundle

- **Development**
  - TypeScript with strict typing
  - Vite build system with rolldown
  - Hot module replacement for development
  - Source maps for debugging

- **Deployment**
  - Production-ready builds
  - Semantic versioning
  - Automated release scripts
  - CDN distribution

### Technical Specifications
- **Bundle Size**: ~45KB gzipped
- **Browser Support**: Chrome 64+, Firefox 63+, Safari 12+, Edge 79+
- **Dependencies**: LitElement (runtime), TypeScript (development)
- **Build System**: Vite with rolldown
- **Package Formats**: ES modules, UMD

### Documentation
- Complete deployment guide
- API reference documentation
- Integration examples
- Troubleshooting guide
- Performance optimization tips
